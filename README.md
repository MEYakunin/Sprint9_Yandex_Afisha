Итоговый проект 2 модуля (Спринт 9) - Анализ пользователей Яндекс Афиши

--Описание проекта--
Анализ поведения пользователей сервиса для выявления факторов, влияющих на повторные покупки. На основе данных о заказах построены профили пользователей и проведено исследование ключевых метрик.

--Ход выполнения--
1. Загрузка и предобработка данных:
    * Выгрузка данных через SQL-запрос к базе данных Яндекс Афиши
    * Приведение выручки к единой валюте (конвертация тенге → рубли по курсу 2024 года)
    * Обработка пропусков и приведение типов данных
    * Фильтрация выбросов по 99-му перцентилю в столбце revenue_rub

2. Построение профилей пользователей:
    * Дата первого и последнего заказа
    * Устройство, регион, билетный партнёр и жанр мероприятия для первого заказа
    * Количество заказов, средняя выручка и среднее количество билетов
    * Среднее время между заказами
    * Бинарные признаки: is_two (2+ заказа), is_five (5+ заказов)

3. Исследовательский анализ:
    * Анализ связи признаков первого заказа с возвращаемостью
    * Проверка гипотез о влиянии типа мероприятия и региона на повторные покупки
    * Исследование зависимости между выручкой/количеством билетов и частотой заказов
    * Анализ временных характеристик (день недели, интервалы между покупками)
    * Корреляционный анализ признаков с числом заказов (коэффициент phi_k)

--Результаты--
    * Определены сегменты пользователей с наибольшей вероятностью возврата
    * Выявлены ключевые факторы, влияющие на повторные покупки
    * Сформулированы рекомендации для отдела маркетинга по удержанию пользователей
